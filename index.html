conceptual-git

git concepts for human beings

Follow @tonyheupel

A video series dedicated to bridging the machine-to-human-being gap when it comes to using git every day.

yet another git talk?

These videos are different. They do not start with the complexity of the internal object store, hashes, or crazy things you can do with git. These videos help you build a conceptual working model of git commits and branches based on other programming concepts most people are familiar with.

The videos explain git concepts using images, animation, text, and actual code. My hope is that presenting the same concepts to audiences using a combination of formats will help to really reinforce the concepts of git.

The content is not necessarily a fully technically correct discription of git internals, but is more of an exploration of the concepts using familiar programming fundamentals.

organization

The videos are intended to be modular and to-the-point. As more videos are added, they will be grouped into playlists by their topic in such a way that you can start from the beginning and build up the concepts you need in order to understand the topic of that playlist. For example, if you wanted to learn more about how 

git rebase -i HEAD~3
works, you would look for the interactive rebase playlist. That playlist would contain:
  • the introductory video to help establish core concepts
  • the video on git rebase
  • and then a video specifically on interactive rebase nuances
